To Do App Deployment

Project information

  • Project date: April 2024
  • Tech used: DevOps, Kubernetes, Google Cloud Platform, Docker
  • GitHub link

Engineered and managed a Kubernetes-based CI/CD pipeline for deploying a scalable web application on GCP, utilizing Docker, GitHub, and Cloud Build for automated workflows. Created development and production environments in a multi-node setup, with multiple pods for high availability, MongoDB database, and a load balancer to distribute incoming traffic.
